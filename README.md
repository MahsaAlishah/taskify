# 📝 Taskify - Todo App

A modern Todo application built with React, Context API, useReducer, React Router, and Tailwind CSS.

---

## 🚀 Features

- Add, edit, delete tasks
- Mark tasks as completed
- Filter tasks (All / Active / Completed)
- Dark / Light theme (saved in localStorage)
- Task statistics dashboard
- Toast notifications
- Responsive design

---

## 🛠️ Tech Stack

React, Context API, useReducer, React Router, Tailwind CSS, React Hot Toast, Vite

---

## ⚙️ Setup

```bash
git clone https://github.com/your-username/taskify.git
cd taskify
npm install
npm run dev
````

---

## 📊 Features Overview

* Persistent state with localStorage
* Global state management with Context + Reducer
* Theme system with system preference support
